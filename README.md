# Maze-Solving Robot

**Biomedical Engineering | University of Sydney | 2024**

## Overview
A fully autonomous maze-solving robot built using C++ and Arduino, independently developed with no prior robotics experience.

## Technical Approach
- Sensor-based maze detection and navigation
- Autonomous pathfinding algorithm implemented in Arduino
- Embedded systems programming in C++

## Repository Contents
- `maze_robot.ino` - Arduino source code
